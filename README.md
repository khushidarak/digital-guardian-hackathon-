# Digital Guardian – AI Fraud Detection System
AI-powered system to detect fraud across email, calls, and transactions in real-time.

## Problem Statement
India lost ₹54,000 Crore to cyber frauds last year.
Current systems work separately:
- Email filters detect phishing  
- Call filters detect spam  
- Banks detect suspicious transactions  
❌ But they don’t connect → scams still happen.

## Solution
We built a Unified Fraud Detection System that:
- Combines multiple data sources  
- Detects fraud patterns  
- Generates a Fraud Risk Score  
- Blocks suspicious transactions in real-time
